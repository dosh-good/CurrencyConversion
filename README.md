# CurrencyConversion
This mini project is written in C++ prompts the user for currency conversion.
Currently there are a total of five conversions: 
US Dollar to CAN, 
US Dollar to EURO, 
EURO to CAN, 
CAN to US Dollar, 
and CAN to EURO.
